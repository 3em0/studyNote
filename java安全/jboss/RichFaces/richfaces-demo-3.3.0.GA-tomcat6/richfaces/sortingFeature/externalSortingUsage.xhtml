In order to customize built-in sorting you just need:

  • Turn selfSorted on the columns to "false"
  • Bind sortOrder attributes on the columns to the bean properties.
  • Manage the bindings with values from org.richfaces.model.Ordering enum.
  • If the sorting is multiple - fill the sortPriorities list with column ids in the order the columns should be sorted.
